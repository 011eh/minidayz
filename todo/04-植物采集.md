## 0. 两套机制（必须分清）

资源虽都叫"植物"，但**行为是两类**，节点骨架共用、Timer 语义不同：

| | **浆果灌木** `berry_bush` | **作物** `pepper/squash/tomato` |
|---|---|---|
| 形态 | 3 种浆果 × 有果/无果 = **6 帧** | 单种、**多生长阶段帧**（原版 0–6 共 7 段 + 枯萎帧） |
| 状态 | 二态循环 `HAS_FRUIT ⇄ EMPTY` | 单向生长 `阶段0→…→成熟(→枯萎)` |
| 触发 | 采集后**定时重生**（Timer 一次性倒计时） | 周期 tick **推进生长**（Timer 重复），成熟才可收 |
| 来源 | 世界固定散布（kind 3 地面物，城镇周边） | 野外散布 或 玩家种植（农田 tile） |
| 收获次数 | 无限（反复长） | 一次（收完销毁 / 或重置回阶段 0） |
| 帧布局 | `hframes=3, vframes=2` | `hframes=阶段数, vframes=1`（**按 PNG 实测核对**） |

> ⚠️ 帧数待核：原版作物文档写 7 阶段，但 `plant_*.png` 实际列数需在建场景时用编辑器核对，以 PNG 为准。

## 1. 节点结构（沿用 `tree.tscn` 范式）

两类共用同一骨架，`InteractionArea` 复用现成基类（`mechanics/interaction_area.gd`），采集脚本仿 `chop_interaction.gd`：

```
HarvestPlant (Node2D)              @tool, class_name, 状态+换帧
├── Sprite (Sprite2D)              hframes/vframes 切帧
├── InteractionArea (实例 mechanics/interaction_area.tscn)
│   └── CollisionShape2D           玩家进范围才可采
│       └─ HarvestInteraction.gd   extends InteractionArea, 重写 _on_interact()
├── Timer                          浆果=重生倒计时 / 作物=生长 tick
├── [可选] PickSound (AudioStreamPlayer2D)
└── （植物无 StaticBody：原版植物无碰撞，不挡路）
```

- **无碰撞**：原版 `objects/object_287/552/554.json` 明确植物无碰撞，玩家可踩过 → **不加 `StaticBody2D`**。
- `InteractionArea` 现成成员：`enabled` / `one_shot` / `interact()` / `_on_interact()`，采集脚本只重写 `_on_interact()`。

## 2. 浆果灌木机制（berry_bush）

二态状态机：`HAS_FRUIT ⇄ EMPTY`。

| 项 | 设计 |
|---|---|
| `@export berry_type: 0/1/2` | 浆果种类，决定帧列 + 产出物品 |
| 帧映射 | sheet 3 列 2 行：有果 `frame = berry_type`（0/1/2），空 `frame = 3 + berry_type`（3/4/5） |
| 采集 `_on_interact` | 给玩家加对应浆果 → 切空帧 → `InteractionArea.enabled=false` → `Timer.start(regrow_time)` |
| `Timer.timeout` | 切回有果帧 → `enabled=true` |
| `@export regrow_time` | 重生冷却，默认参考原版 `LOOT_RESPAWN_TIME≈1480s`，做成 Inspector 可调 |

**浆果物品（原版 ID）**：蔓越莓 `Cranberry` ID27、云莓 `Cloudberry` ID73、越橘 `Lingonberry` ID74。
**毒莓**：原版**未发现**，不做。
**远处随机化**：原版"距玩家≥500 时 `choose(0,1)` 随机帧"是省状态写法，本项目用 Timer 精确重生即可，**不复刻**。

## 3. 作物机制（pepper / squash / tomato）

带生长进度，成熟才可收获一次。

| 项 | 原版数值 / 设计 |
|---|---|
| 生长 tick | 每 **20s** 一次（`Timer` 重复触发） |
| 成熟总时长 | **1440s（24 分钟）** |
| 阶段 | 7 段，帧 0–6；`帧 = floor(进度/240)`，进度每 tick +20 |
| 未成熟 | `InteractionArea.enabled=false`，不可收 |
| 枯萎（可选） | 进度到 **540** 时 **25%** 概率枯萎→枯萎帧、不可收；原版「农艺天赋 perk_agronom」可免疫（天赋系统未做则**先略**） |
| 成熟 | `enabled=true`，可收 |
| 收获 `_on_interact` | 给玩家加产出 → 销毁 或 重置回阶段 0 重长 |
| 种植（可选） | 种子物品在农田 tile 上 `instantiate()` 本场景，从阶段 0 起 |

**作物对应（原版 ID）**：

| 作物 | 对象号 | 种子 ID | 产出物品 |
|---|---|---|---|
| 番茄 tomato | t554 | 47 | Tomato ID5 |
| 辣椒 pepper | t552 | 48 | Bell pepper ID32 |
| 西葫芦 squash | ◻️未确认 | 68 | Zucchini ID31 |

> ◻️ 待核：西葫芦对象号、各作物收获**产量数值**（文档未明文 `iv[1]`）。
